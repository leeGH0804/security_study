
# Suricata

Suricata는 Open Information Security Foundation(OISF)에서 개발·유지하는 고성능 오픈소스 네트워크 위협 탐지 엔진으로, 실시간 IDS(침입 탐지), IPS(침입 예방), NSM(네트워크 보안 모니터링) 기능을 모두 지원합니다.

공식 홈페이지 : https://suricata.io/
