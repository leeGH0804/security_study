
# HIDS - OSSEC

OSSEC는 호스트 기반 침입 탐지와 자동 대응 기능을 모두 갖춘 오픈소스 HIDS로, 멀티 플랫폼 지원과 컴플라이언스 대응이 가능한 강력한 보안 솔루션입니다.

공식 홈페이지 : https://www.ossec.net/
