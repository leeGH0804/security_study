# 덤프 대상

![[메모리 1.jpg|500]]
## SPI Flash 메모리 (MD25Q64CSIG)

- **역할:** 공유기의 펌웨어를 저장하는 비휘발성 메모리
- **인터페이스:** SPI (Serial Peripheral Interface)
- **용량:** 64Mbit (8MB)
- **저장 내용:** Bootloader, Kernel, Root File System, 환경설정
- **메모리 덤프 대상:** Flashrom을 이용하여 내용을 `.bin` 파일로 추출

# 덤프 도구
## 1. SOIC8 클립

![[SOIC8 클립.jpg|500]]
### 역할

- SPI Flash 메모리에 직접 연결하여 데이터를 읽기 위한 클립
- 칩을 기판에서 분리하지 않고 메모리 덤프 가능

### 특징

- 8핀 SPI Flash에 연결
- 납땜 없이 메모리 접근 가능
- CH341A/B 프로그래머와 연결하여 사용

## 2. SOP16/8 → DIP8 어댑터

![[어댑터.jpg|500]]

### 역할

- SOIC8 클립과 CH341A를 연결하는 변환 어댑터
- 핀 배열을 CH341A에서 사용할 수 있도록 변환

### 특징

- SOIC8 ↔ DIP8 변환
- SPI Flash 연결 편의성 향상
- CH341A ZIF 소켓에 쉽게 연결 가능

## 3. CH341A USB 프로그래머

![[ch341a.jpg|500]]

### 역할

- SPI Flash와 PC를 연결하는 USB 프로그래머
- Flashrom을 이용하여 메모리 읽기 및 쓰기 수행

### 특징

- USB 인터페이스 지원
- SPI Flash 읽기/쓰기 가능
- Flashrom 등 다양한 프로그램 지원

## 3. Flashrom으로 메모리 덤프

![[펌웨어 추출.png|500]]

**설명**

- Flashrom 실행
- SPI Flash 칩 인식
- 메모리 내용을 `.bin` 파일로 저장

명령어

```
# CH341A 프로그래머를 통해 SPI Flash 메모리 정상 인식 여부를 확인
sudo flashrom -p ch341a_spi

# SPI Flash 메모리의 내용을 읽어 `XXXXXX_dump_1.bin` 파일로 저장
sudo flashrom -p ch341a_spi -r XXXXXX_dump_1.bin
```
