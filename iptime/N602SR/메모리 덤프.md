## SPI Flash 메모리 (MD25Q64CSIG)

- **역할:** 공유기의 펌웨어를 저장하는 비휘발성 메모리
- **인터페이스:** SPI (Serial Peripheral Interface)
- **용량:** 64Mbit (8MB)
- **저장 내용:** Bootloader, Kernel, Root File System, 환경설정
- **메모리 덤프 대상:** Flashrom을 이용하여 내용을 `.bin` 파일로 추출

- SPI Flash 메모리에 직접 연결하여 데이터를 읽기 위한 클립

## 3. Flashrom으로 메모리 덤프

### 사전 작업

⭐ CH341A 장치 인식 확인
- Windows 장치관리자에서 CH341A가 정상적으로 인식되는지 확인
- 인식되지 않을 경우 CH341A 드라이버 설치 후 재확인

⭐ WSL 환경에서 USB 연결
- WSL에서 Flashrom을 사용하려면 CH341A를 WSL에 연결해야 함
- usbipd를 이용하여 USB 장치를 Attach
- lsusb 명령어로 정상 인식 여부 확인

#### Windows (관리자 권한)
```
usbipd list
usbipd bind --busid <BUSID>
usbipd attach --wsl --busid <BUSID>
```

#### WSL
```
lsusb
```
CH341A 장치가 목록에 표시되면 정상적으로 연결된 상태

### flashrom 으로 펌웨어 추출

<img width="976" height="350" alt="펌웨어 추출" src="https://github.com/user-attachments/assets/35005efd-02bb-49a3-a57d-7b021da6926e" />

**설명**

- Flashrom 실행
- SPI Flash 칩 인식
- 메모리 내용을 `.bin` 파일로 저장

명령어

```
# CH341A 프로그래머를 통해 SPI Flash 메모리 정상 인식 여부를 확인
sudo flashrom -p ch341a_spi

# SPI Flash 메모리의 내용을 읽어 `XXXXXX_dump_1.bin` 파일로 저장
sudo flashrom -p ch341a_spi -r XXXXXX_dump_1.bin
```
